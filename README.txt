3.Svolgimento
Per avviare AlGraph cliccare l'eseguibile (AlGraph.jar).
All'avvio l'utente potrà decidere fra 3 azioni:
-importare il grafo da un file.txt esterno
-generare un grafo casuale
-costruire il grafo da zero
In seguito, l'utente potrà modificare, aggiungere o eliminare archi e nodi
nell'editor del grafo, esportare il grafo su un file .txt esterno o eseguire
l'algoritmo dei cammini.
Una volta azionato l'algoritmo, l'utente potrà seguire l'esecuzione passo passo
o visualizzare direttamente il risultato.

Per la corretta visualizzazione del font è necessario installarlo tramite
il file simplifica.ttf nella cartella font (Algraph\Consegna\Algraph\src\font\SIMPLIFICA).
